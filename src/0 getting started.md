# 0. Getting started

## Installing

`brew install deno`
`curl -fsSL https://deno.land/x/install/install.sh | sh`
`choco install deno`


## Upgrading

`deno upgrade`


## Autocompletion

```zsh
mkdir ~/.oh-my-zsh/custom/plugins/deno
deno completions zsh > ~/.oh-my-zsh/custom/plugins/deno/_deno

# add deno as plugin

```
